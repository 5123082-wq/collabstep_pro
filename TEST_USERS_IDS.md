## Использование в коде

```typescript
import { TEST_ADMIN_USER_ID, TEST_USER_ID } from '@collabverse/api';

// Использование ID администратора
const adminId = TEST_ADMIN_USER_ID; // '00000000-0000-0000-0000-000000000001'
```

## Обратная совместимость

- `DEFAULT_WORKSPACE_USER_ID` теперь равен `TEST_ADMIN_USER_ID`
- Функция `isAdminUserId()` проверяет оба значения для обратной совместимости
- Старые сессии с email в качестве userId будут работать благодаря fallback в `decodeDemoSession()`

## Важно

**ID администратора для тестирования**: `00000000-0000-0000-0000-000000000001`

Этот ID используется во всех тестах и демо-данных. При создании новых тестов используйте константу `TEST_ADMIN_USER_ID` вместо хардкода.
