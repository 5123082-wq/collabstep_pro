# Marketing Glow Effect

## Описание
Эффект градиентной подсветки для блоков маркетингового раздела. Создает внешнее свечение с градиентом индиго/фиолетового цвета, придавая блокам визуальную глубину и акцент.

## Технические детали
Эффект реализован через CSS класс `.marketing-glow`, который включает:
- Многослойную тень (`box-shadow`) с градиентом от черного к индиго
- Псевдоэлемент `::before` с градиентной рамкой для внешнего свечения
- Адаптацию для светлой и тёмной темы

## Дизайн-токен
Токен `glow-shadow` определен в `design-tokens.ts`:
```typescript
'glow-shadow': '0_0_30px_rgba(0,0,0,0.25), 0_0_60px_rgba(99,102,241,0.1), 0_0_90px_rgba(99,102,241,0.05)'
```

## Использование
Применяйте класс `marketing-glow` к основным блокам в маркетинговом разделе:

```tsx
// Основной блок секции
<section className="marketing-glow space-y-6 rounded-3xl border border-neutral-900 bg-neutral-950/70 p-6">
  {/* содержимое */}
</section>

// Карточка маркетинга (уже применено в MarketingCard)
<MarketingCard title="Заголовок">
  {/* содержимое */}
</MarketingCard>

// Блок метрик (уже применено в MarketingHeader)
<div className="marketing-glow rounded-2xl border border-neutral-900 bg-neutral-950/70 p-4">
  {/* метрика */}
</div>
```

## Где применено
- ✅ Все компоненты `MarketingCard` - автоматически применяется
- ✅ Блоки метрик в `MarketingHeader` - автоматически применяется
- ✅ Основной блок "Маркетинговый хаб" в `overview/page.tsx`

## Применение к новым блокам
При создании новых блоков в маркетинговом разделе или других разделах портала, добавьте класс `marketing-glow` к основному контейнеру:

```tsx
<div className="marketing-glow rounded-3xl border border-neutral-900 bg-neutral-950/60 p-6">
  {/* ваш контент */}
</div>
```

## Адаптация для других разделов
Класс можно использовать в любом разделе портала для создания эффекта градиентной подсветки. Просто добавьте класс `marketing-glow` к нужному элементу.
