## Введение

Управление промптами — это создание и использование шаблонов промптов для различных задач. Функциональность планируется.

## Структура промпта

```typescript
interface AIPrompt {
  id: string;
  name: string;
  description: string;
  template: string; // Шаблон промпта с переменными
  variables: AIPromptVariable[]; // Переменные в шаблоне
  category: string; // Категория промпта
  tags: string[]; // Теги для поиска
  createdBy: string; // ID создателя
  createdAt: string;
  updatedAt: string;
}
```

## Переменные в промптах

Промпты могут содержать переменные:
- `{taskTitle}` — название задачи
- `{projectName}` — название проекта
- `{userName}` — имя пользователя
- `{context}` — контекст запроса

## Создание промпта

1. Пользователь открывает раздел "Промпты"
2. Нажимает "Создать промпт"
3. Заполняет форму:
   - Название промпта
   - Описание
   - Шаблон промпта (с переменными)
   - Категория и теги
4. Сохраняет промпт

## Использование промптов

Промпты используются в:
- Генерации контента
- AI-агентах
- Автоматизации задач

### Пример использования

**Промпт:**
```
Создай описание задачи "{taskTitle}" для проекта "{projectName}".
Включи цели, задачи и критерии готовности.
```

**Результат:**
Промпт заполняется переменными и отправляется в AI для генерации.

## Шаблоны промптов

Платформа предоставляет готовые шаблоны промптов:
- Генерация описания задачи
- Суммирование комментариев
- Планирование проекта
- Анализ загруженности
