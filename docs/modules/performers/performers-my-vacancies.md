# Исполнители — Мои вакансии

**Статус:** partial
**Владелец:** product/engineering
**Создан:** 2026-01-07
**Последнее обновление:** 2026-01-15

## Назначение

Раздел «Мои вакансии» показывает список вакансий, созданных организацией, и статусы откликов.

## Навигация

- `/performers/my-vacancies` — мои вакансии

## Текущее состояние (по UI и коду)

- Список вакансий строится из API `/api/vacancies` с фильтром по `organizationId`.
- Доступен фильтр статуса: `all`, `draft`, `published`, `closed`.
- Создание вакансии выполняется через модальное окно, доступное из кнопки «+ Создать».
- Переход к карточке вакансии ведет на `/performers/vacancies/:id`, где доступны редактирование и вложения.

## Основные сценарии

1. Создать вакансию и опубликовать в каталоге.
2. Отфильтровать список по статусу.
3. Открыть карточку вакансии для редактирования и работы с вложениями.

## Создание

1. Нажать кнопку «+ Создать» (в разделе или в глобальном меню).
2. Заполнить форму вакансии (часовой пояс по умолчанию берется из профиля, его можно изменить).
3. Прикрепить файлы при создании — загрузка идет через Vercel Blob.
4. После сохранения вакансия появится в списке.

## Статусы и роли

- Статусы вакансий: `draft` | `published` | `closed`.
- Создавать и редактировать вакансии могут только owner/admin организаций.

## Доступность по ролям

- Управление вакансиями — owner/admin организаций.

## Взаимодействия с другими разделами

- **Вакансии и задачи:** публикация вакансий для каталога.
- **Отклики и приглашения:** обработка откликов.

## Ограничения и планируемое

- Для загрузки файлов требуется настроенный `BLOB_READ_WRITE_TOKEN`.
