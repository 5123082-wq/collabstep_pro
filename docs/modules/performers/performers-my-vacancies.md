# Исполнители — Мои вакансии

**Статус:** partial
**Владелец:** product/engineering
**Создан:** 2026-01-07
**Последнее обновление:** 2026-01-07

## Назначение

Раздел «Мои вакансии» должен показывать список вакансий, созданных организацией, и статусы откликов.

## Навигация

- `/performers/my-vacancies` — мои вакансии

## Текущее состояние (по UI и коду)

- Страница содержит сценарные действия без реального списка вакансий.

## Планируемые сценарии

1. Создание вакансии и публикация в каталоге.
2. Управление статусами вакансии (черновик/опубликовано/закрыто).
3. Просмотр откликов и приглашений.

## Создание

Планируется запуск мастера создания вакансии. В текущем UI доступна только заглушка.

## Спорные статусы и роли

- Статусы вакансий и откликов будут уточнены после подключения API.
- Создавать вакансии смогут owner/admin организаций.

## Доступность по ролям

- Управление вакансиями — owner/admin организаций.

## Взаимодействия с другими разделами

- **Вакансии и задачи:** публикация вакансий для каталога.
- **Отклики и приглашения:** обработка откликов.

## Ограничения и планируемое

- API вакансий отсутствует.
