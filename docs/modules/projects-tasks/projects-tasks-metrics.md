## Введение

Дашборд workspace (`/pm`) предоставляет единую точку входа для обзора всех проектов, задач, метрик и активности. Дашборд содержит настраиваемые виджеты с ключевыми метриками и быстрыми действиями. Все виджеты кликабельны и ведут к детальным представлениям (дрилдауны).

**Статус реализации:** ⚠️ Частично реализован

## Виджеты

### Пульс (Activity Pulse)

**Описание:** Отображает активность проектов за последние 7 дней.

**Метрики:**
- Количество созданных задач
- Количество обновленных задач
- Количество комментариев
- Количество завершенных задач

**Дрилдаун:**
- Клик по виджету → переход к списку активных проектов
- Фильтр по активности

### Прогресс (Progress)

**Описание:** Отображает общий прогресс по всем проектам.

**Метрики:**
- Процент завершенных задач
- Количество активных проектов
- Количество завершенных проектов
- Средний прогресс по проектам

**Дрилдаун:**
- Клик по виджету → переход к списку проектов с фильтром по прогрессу

### Нагрузка исполнителей (Workload)

**Описание:** Отображает распределение нагрузки между исполнителями.

**Метрики:**
- Количество задач на каждого исполнителя
- Перегруженные исполнители (выделены)
- Исполнители с низкой нагрузкой
- График распределения нагрузки

**Дрилдаун:**
- Клик по виджету → переход к Kanban доске с фильтром по исполнителю
- Детальный просмотр нагрузки конкретного исполнителя

### Финансы (Finance)

**Описание:** Отображает финансовые метрики проектов.

**Метрики:**
- Общий бюджет всех проектов
- Потраченные средства
- Остаток бюджета
- Проекты с превышением бюджета (выделены)

**Дрилдаун:**
- Клик по виджету → переход к финансовому модулю
- Детальный просмотр бюджета проектов

**См. также:**
- [`../../modules/finance/finance-overview.md`](../../modules/finance/finance-overview.md) — документация по финансам

## Дрилдауны

### Переходы к детальным представлениям

Все виджеты кликабельны и ведут к соответствующим разделам:

- **Пульс** → список активных проектов
- **Прогресс** → список проектов с фильтром по прогрессу
- **Нагрузка** → Kanban доска с фильтром по исполнителю
- **Финансы** → финансовый модуль

### Фильтры и контекст

При переходе из виджета:
- Применяются соответствующие фильтры
- Сохраняется контекст (дата, проект, исполнитель)
- Можно вернуться к дашборду

## Состояния виджетов

### Content

Виджет отображает данные:
- Метрики и графики
- Список элементов
- Статус обновления (lastUpdated)

### Loading

Виджет загружает данные:
- Skeleton loader
- Индикатор загрузки
- Сообщение "Загрузка..."

### Empty

Виджет не содержит данных:
- Иконка пустого состояния
- Сообщение "Нет данных"
- CTA для создания данных (если применимо)

### Error

Виджет не может загрузить данные:
- Сообщение об ошибке
- Кнопка "Повторить"
- Информация об источнике ошибки

## API endpoints

**Статус:** ⚠️ Планируется

В будущем планируется добавить API endpoints для дашборда:

- `GET /api/pm/dashboard` — получить данные для всех виджетов
- `GET /api/pm/dashboard/widgets/[type]` — получить данные для конкретного виджета
- `POST /api/pm/dashboard/layout` — сохранить layout дашборда

- `POST /api/pm/dashboard/presets` — создать пресет
