# Маркетплейс — Избранное

**Создан:** 2026-01-07
**Последнее обновление:** 2026-01-07

## Назначение

Избранное хранит подборку шаблонов, к которым пользователь хочет быстро возвращаться. Список используется для повторного доступа и добавления в корзину.

## Навигация

- `/market/favorites` — избранное

## Текущее состояние (по UI и коду)

- Список избранного хранится в `useMarketplaceStore` (Zustand).
- В избранное добавляют из каталога и карточки шаблона.
- Страница показывает карточки шаблонов с кнопками «В корзину» и «Удалить».
- При пустом списке отображается заглушка с переходом в каталог.

## Основные сценарии

1. Добавить шаблон в избранное из каталога или карточки.
2. Удалить шаблон из избранного.
3. Перейти к карточке шаблона.
4. Добавить шаблон в корзину.

## Доступность по ролям

- Раздел доступен ролям `FOUNDER`, `PM`, `ADMIN`, `SPECIALIST`, `CONTRACTOR`.

## Взаимодействия с другими разделами

- **Каталог шаблонов:** основной источник добавления.
- **Корзина:** быстрый перенос выбранных шаблонов к покупке.

## Ограничения и планируемое

- Избранное не синхронизируется с сервером и не сохраняется между сессиями.
