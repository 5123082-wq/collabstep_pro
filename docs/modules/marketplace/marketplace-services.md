# Маркетплейс — Пакеты услуг

**Статус:** partial
**Владелец:** product/engineering
**Создан:** 2026-01-07
**Последнее обновление:** 2026-01-07

## Назначение

«Пакеты услуг» — будущий каталог фиксированных предложений от студий и команд (дизайн, разработка, маркетинг). Раздел будет поддерживать запросы и согласование условий.

## Навигация

- `/market/services` — пакеты услуг

## Текущее состояние (по UI и коду)

- Страница `/market/services` отображает заглушку «в разработке».
- В меню «Создать» для маркетплейса предусмотрено действие «Услуга» (модальный сценарий пока не реализован).

## Планируемые сценарии

1. Просмотр пакетов услуг и карточек.
2. Запрос услуги (inquiry) и согласование условий.
3. Создание заказа после согласования.

## Спорные статусы и роли

- Статусы услуг и запросов будут уточнены отдельно. Для спорных кейсов планируется участие администратора/поддержки.

## Доступность по ролям

- Раздел доступен ролям `FOUNDER`, `PM`, `ADMIN`, `SPECIALIST`, `CONTRACTOR`.

## Взаимодействия с другими разделами

- **Исполнители:** источник провайдеров услуг.
- **Финансы:** оплату и расчёты предстоит интегрировать.

## Ограничения и планируемое

- UI раздела в разработке, API и модели услуг не реализованы.
