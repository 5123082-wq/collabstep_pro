## Введение

Интеграции с внешними платформами позволяют синхронизировать данные из Google Ads, Facebook Ads и других маркетинговых платформ.

NEEDS_CONFIRMATION: UI-заглушки есть, данные и интеграции не реализованы.

## Google Ads

### Подключение интеграции

1. Пользователь открывает настройки интеграций
2. Выбирает "Google Ads"
3. Авторизуется через Google
4. Предоставляет доступ к аккаунтам Google Ads
5. Выбирает аккаунты и кампании для синхронизации
6. Сохраняет интеграцию

NEEDS_CONFIRMATION: Детали интеграции с Google Ads. Какие данные синхронизируются?

### Синхронизация данных

- Метрики кампаний
- Расходы
- Клики и показы
- Конверсии

## Управление интеграциями

### Отключение интеграции

Пользователи могут отключить интеграцию:
1. Открывают настройки интеграций
2. Выбирают интеграцию
3. Нажимают "Отключить"
4. Подтверждают отключение

### Обновление данных

Пользователи могут вручную обновить данные:
- Кнопка "Обновить данные" в настройках интеграции
- Автоматическое обновление по расписанию
