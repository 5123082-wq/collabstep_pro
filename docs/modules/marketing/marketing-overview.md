## 1) Purpose

Marketing — модуль маркетинговой аналитики и интеграций платформы Collabverse. Модуль предоставляет дашборды с метриками ROI/CPC/CPA/CLV, управление кампаниями, интеграции с внешними платформами и экспорт отчетов.

**Роль в платформе:**

- Маркетинговая аналитика и отчетность
- Управление маркетинговыми кампаниями
- Интеграции с Google Ads, Facebook Ads и другими платформами
- Экспорт и публикация отчетов

NEEDS_CONFIRMATION: UI-разделы есть, данные и интеграции не реализованы.

## 2) Key objects

Основные сущности модуля Marketing:

- **Dashboard** — маркетинговый дашборд с виджетами метрик
- **Campaign** — маркетинговая кампания
- **Report** — отчет по маркетинговой активности
- **Integration** — интеграция с внешней платформой (Google Ads, Facebook Ads)

## 3) Top user scenarios (E2E)

### Сценарий 1: Просмотр маркетингового дашборда

1. Пользователь открывает раздел "Маркетинг"
2. Видит дашборд с метриками ROI, CPC, CPA, CLV
3. Просматривает виджеты с данными по кампаниям
4. Фильтрует данные по периоду и источникам

### Сценарий 2: Создание кампании

1. Пользователь создает новую кампанию
2. Заполняет параметры кампании (название, бюджет, сроки)
3. Выбирает источники трафика
4. Запускает кампанию
5. Отслеживает метрики кампании

2. Авторизуется через Google
3. Выбирает аккаунты и кампании для отслеживания
4. Данные синхронизируются с платформой
