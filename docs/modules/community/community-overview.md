## 1) Purpose

Community — модуль сообщества платформы Collabverse. Модуль предоставляет питчи, комнаты и взаимодействие с сообществом.

**Роль в платформе:**

- Питчи проектов и идей
- Комнаты для обсуждений
- Взаимодействие с сообществом
- Обмен опытом и знаниями

## 3) Top user scenarios (E2E)

### Сценарий 1: Создание питча

1. Пользователь создает питч проекта
2. Заполняет описание проекта (название, идея, команда)
3. Добавляет презентацию или видео
4. Публикует питч
5. Сообщество может комментировать и оценивать питч

### Сценарий 2: Участие в комнате

1. Пользователь находит комнату по интересам
2. Присоединяется к комнате
3. Участвует в обсуждениях
4. Делится опытом и знаниями

### Сценарий 3: Взаимодействие с сообществом

1. Пользователь просматривает активность сообщества
2. Комментирует питчи и обсуждения
3. Оценивает проекты и идеи
4. Связывается с другими участниками
