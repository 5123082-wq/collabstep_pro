## Описание

Полный реестр всех документов в `/docs` с метаданными, происхождением и связями.

## Легенда

- **Doc ID**: стабильный идентификатор документа
- **Происхождение**: `NEW` - создан в новой структуре, `MIGRATED` - мигрирован из старой структуры
- **Статус**: `draft` / `review` / `stable` / `active` / `deprecated` / `archive`
- **Confidence**: `high` - документ соответствует реальности, `medium` - частично соответствует, `low` - требует проверки
