# План миграции типографики

## Цель

Привести все компоненты к единой системе типографики с минимальным набором размеров шрифтов.

---

## Шаг 1: Замена нестандартных размеров

### 1.1. Замена `text-[10px]` на `text-xs`

**Файлы для проверки:**
- `components/pm/ProjectCardTile.tsx`
- `components/pm/ProjectTeam.tsx`
- Другие PM компоненты

**Действие:**
```tsx
// ❌ Было
<span className="text-[10px]">Статус</span>

// ✅ Стало
<span className="text-xs">Статус</span>
```

---

## Шаг 2: Стандартизация заголовков блоков

### 2.1. Заголовки ContentBlock → `text-lg`

Все заголовки внутри `ContentBlock` должны использовать `text-lg` (или `ContentBlockTitle` компонент).

**Файлы для проверки:**
- `components/pm/ProjectHeader.tsx` - проверить использование заголовков
- `components/app/AppSection.tsx` - заменить `text-xl` на `text-lg` для заголовков блоков
- Все компоненты с заголовками внутри блоков

**Правило:**
- Заголовки **блоков контента** → `text-lg`
- Заголовки **разделов/страниц** → `text-xl`

---

## Шаг 3: Стандартизация основного текста

### 3.1. Замена `text-base` на `text-sm` в блоках контента

В основном контенте приложения использовать `text-sm` вместо `text-base`.

**Исключения:**
- Компоненты маркетинга могут использовать `text-base` для подзаголовков
- Body уже использует 16px (base), не нужно явно указывать `text-base`

**Действие:**
```tsx
// ❌ Было (если в блоке контента)
<p className="text-base">Текст</p>

// ✅ Стало
<p className="text-sm">Текст</p>
```

---

## Шаг 4: Ограничение больших размеров

### 4.1. `text-2xl` и `text-3xl` только для маркетинга

В основном контенте приложения использовать `text-xl` вместо `text-2xl`/`text-3xl`.

**Разрешено использовать `text-2xl`/`text-3xl`:**
- `components/marketing/sections/Hero.tsx`
- `components/marketing/sections/Features.tsx`
- `components/marketing/sections/Audience.tsx`
- `components/marketing/sections/CTA.tsx`
- `components/marketing/app/MarketingHeader.tsx`

**Запрещено использовать `text-2xl`/`text-3xl`:**
- PM компоненты
- App компоненты (кроме маркетинга)
- Marketplace компоненты
- UI компоненты (кроме маркетинга)

**Действие:**
```tsx
// ❌ Было (в основном контенте)
<h2 className="text-3xl">Заголовок</h2>

// ✅ Стало
<h2 className="text-xl">Заголовок</h2>

// ✅ Или если нужен действительно большой заголовок (редко)
<h1 className="text-xl font-semibold">Заголовок страницы</h1>
```
