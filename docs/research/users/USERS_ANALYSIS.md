# Анализ пользователей в системе

## Источники пользователей

Пользователи в админ-панели берутся из двух источников:
1. `WORKSPACE_USERS` - основная таблица пользователей
2. `ADMIN_USER_CONTROLS` - настройки доступа пользователей

Метод `listUsers()` объединяет оба источника, поэтому если пользователь есть в `ADMIN_USER_CONTROLS`, но не в `WORKSPACE_USERS`, он все равно появится в списке.
